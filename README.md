# ia-data-cleaning-notebooklm
Projeto de estudo sobre IA aplicada ao tratamento e limpeza de dados (Data Cleaning), com uso de NotebookLM para organização de fontes, engenharia de prompts e criação de guia de aprendizagem.
Este projeto tem como objetivo explorar o uso da Inteligência Artificial no processo de tratamento e limpeza de dados (Data Cleaning), utilizando o NotebookLM como ferramenta de apoio para organização e síntese do conhecimento.

### Objetivos principais:
- Entender como a IA auxilia na limpeza e preparação de dados
- Identificar técnicas de tratamento de dados inconsistentes
- Explorar automação de tarefas repetitivas em datasets
- Criar um miniguia de estudo estruturado sobre Data Cleaning com IA

##🤖 Engenharia de Prompts e “Cicatrizes”:

### 🔎 Prompt 1:
"Explique o que é data cleaning de forma simples e com exemplos práticos"
**Resposta resumida:**
"Limpeza de dados (ou data cleaning) é o processo de identificar e corrigir erros, inconsistências e imprecisões em conjuntos de dados brutos para melhorar sua qualidade
. De forma simples, é uma "faxina" sistemática em uma base de dados para remover informações incorretas, corrompidas, mal formatadas, duplicadas ou incompletas, garantindo que o resultado final seja confiável para análise ou tomada de decisão"

### 🔎 Prompt 2:
"Quais são as principais técnicas de limpeza de dados usadas em projetos de IA?"
**Resposta resumida:**
"Tratamento de Dados Ausentes (Missing Values), Remoção de Duplicatas (Deduplicação), Padronização e Polimento de Dados, Tratamento de Outliers (Valores Discrepantes), Técnicas Avançadas com IA e NLP, Transformação e Seleção de Atributos (Feature Engineering)."

### 🔎 Prompt 3:
"Como a IA pode automatizar o processo de limpeza de dados?"
**Resposta resumida:**
"A inteligência artificial (IA) automatiza a limpeza de dados ao substituir regras rígidas e manuais por algoritmos que aprendem padrões, identificam anomalias e aplicam correções de forma adaptativa
Enquanto métodos tradicionais dependem de instruções fixas, a IA observa como os dados "limpos" devem ser e melhora sua precisão à medida que processa mais exemplos e recebe feedback humano"

## 📖 Miniguia de Estudo (Resultado Final)

### 🧠 O que é Data Cleaning
Data Cleaning é o processo de preparação de dados para análise, removendo erros, duplicações e inconsistências.

---

### ⚙️ Principais técnicas
- Remoção de duplicados
- Tratamento de valores ausentes
- Normalização de dados
- Padronização de formatos
- Identificação de outliers

---

### 🤖 Como a IA ajuda no Data Cleaning
- Detecta padrões automaticamente
- Sugere correções em dados inconsistentes
- Preenche valores ausentes com base em previsões
- Automatiza processos repetitivos

---

### 📌 Aplicações práticas
- Análise de dados empresariais
- Machine Learning
- BI (Business Intelligence)
- Relatórios automatizados

---

### 📖 Glossário dos principais conceitos aprendidos
- **Data Cleaning:** processo de limpeza e correção de dados
- **Outlier:** valor fora do padrão esperado
- **Dataset:** conjunto de dados
- **Imputação:** técnica de preenchimento de valores ausentes
- **Normalização:** ajuste de escala dos dados

---
